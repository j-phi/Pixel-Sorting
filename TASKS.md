# Tasks

Feature roadmap and planned work for Pixel Sort Studio.

## Modes

- [ ] **Auto mode** — press a button and the action is continuously reapplied until the next button press
- [ ] **Random mode** — randomly generate steps of a user-defined length and apply them
- [ ] **Inference mode** — generate a random sorting path between a fixed input image and an output image

## Interactivity

- [ ] **Canvas interactions** — mouse click, drag, and scroll wheel to control sorting region, brush size, or threshold
- [ ] **Start from uploaded image** — load a pre-sorted image as a starting point

## UX / Education

- [ ] **Tooltips** — hover explanations for every control
- [ ] **Explainer section** — inline guide explaining what pixel sorting is, how masks work, etc.
- [ ] **Tutorial mode** — guided walkthrough for first-time users

## Export & Persistence

- [ ] **Export playback as GIF** — capture the history stack as an animated GIF
- [ ] **Save sort steps as JSON** — serialize the recorded macro into a reloadable `.json` file
- [ ] **Load sort steps from JSON** — import a previously saved macro and replay it
